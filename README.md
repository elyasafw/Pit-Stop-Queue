# Pit-Stop-Queue

מערכת קטנה לניהול תור עצירות פיט בזמן מרוץ. המערכת מושכת נתוני מרוץ ורכבים משרת, ומציגה דוח חי על מצב התור - אילו רכבים ממתינים לפיט, מי הבא בתור, וחיפוש רכב לפי מספר.

## קבצים עיקריים

- `serverData.js` - שליפת נתונים מהשרת ושמירתם מקומית ב-`data.json`, וקריאה מהקובץ המקומי.
- `race.js` - מידע כללי על המרוץ (שם המרוץ, ליפ נוכחי, סה"כ ליפים).
- `cars.js` - מידע על הרכבים: רשימת רכבים, חיפוש רכב לפי מספר, סינון רכבים הממתינים לפיט.
- `main.js` - נקודת הכניסה של התוכנית, מריצה את כל התהליך ומדפיסה את דוח התור.

## הרצה

```bash
npm install
node main.js
```

בכל הרצה התוכנית מושכת נתונים עדכניים מהשרת, שומרת אותם ב-`data.json`, ואז מדפיסה דוח על תור עצירות הפיט הנוכחי.
